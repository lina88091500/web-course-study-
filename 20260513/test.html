內容 (Content)

由外而內：Margin ➔ Border ➔ Padding ➔ Content

2. Margin vs. Padding 的終極差別

🌟 Padding (留白)： 它是盒子的「肉」。當你設定背景顏色時，Padding 區域會被填滿顏色。它是用來讓內容不要撞到邊框。

🌟 Margin (邊界)： 它是盒子的「空氣」。它是透明的，用來把別的元素推開，不讓大家擠在一起。

3. 那些讓你頭痛的縮寫 (TM, LP...)

縮寫首字 代表方位 範例
T (Top) 上方 TM = 上邊界 / TP = 上留白
B (Bottom) 下方 BM = 下邊界 / BP = 下留白
L (Left) 左方 LM = 左邊界 / LP = 左留白
R (Right) 右方 RM = 右邊界 / RP = 右留白

4. 為什麼網頁會變寬？(計算公式)

講義範例：內容 60px，留白/框線/邊界各為 8/4/8px。

計算方式：60 + (8+4+8) × 2 = 100px

💡 記得要 × 2，是因為左右兩邊都有這些距離喔！

5. 邊界重疊 (Margin Collapse)

這是最容易考的地方！當兩個垂直的 Margin 碰在一起，它們不會相加，而是會像大魚吃小魚一樣，只留下最大的那個數字。

範例：上面盒子想留 20px，下面盒子想留 10px，結果中間只會隔 20px。
